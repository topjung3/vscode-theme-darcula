# Darcula Heesu Theme

A VS Code color theme based on Darcula theme from JetBrains IDEs.

## Installation

### Option 1: Install Script

```bash
chmod +x install.sh
./install.sh
```

### Option 2: Manual Installation

Copy the extension folder to VS Code extensions directory:

```bash
cp -r . ~/.vscode/extensions/vscode-theme-darcula-heesu-0.0.1/
```

## Activating the Theme

1. Restart VS Code
2. Open Command Palette: `Cmd+K Cmd+T` (macOS) or `Ctrl+K Ctrl+T` (Linux/Windows)
3. Select **Darcula-heesu**

## Build

No build step required. This is a pure JSON theme extension.
